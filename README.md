  核心思路
                                                                                
  矩阵乘法（GEMM）是HPC领域的"Hello                                             
  World"，同时也是面试官最熟悉的考察点，覆盖了路线中从 Easy 到 Lunatic
  的所有核心技术。

  ---
  简历描述示例

  高性能矩阵乘法多层优化 | C/C++, CUDA, AVX2, OpenMP  [2025.xx]
  - 从朴素 O(n³) 实现出发，通过循环重排、缓存分块将 Cache Miss 率降低 ~80%
  - 使用 AVX2 SIMD 手动向量化内核，结合 FMA 指令达到 ~85% 峰值 FLOPS
  - 引入 OpenMP 实现多核并行，4 核下获得 3.7x 线性加速比
  - CUDA 实现 Shared Memory Tiling，GPU 上吞吐量达 OpenBLAS 的 ~70%
  - 使用 Linux perf / Intel VTune 定位热点，输出性能分析报告

  ---
  为什么选这个项目？

  1. 面试高频考点 —— 几乎每个HPC岗都会问矩阵乘法优化
  2. 技术栈完整 —— 覆盖SIMD、多线程、CUDA、性能分析工具，一个项目打通路线全程
  3. 可量化 —— 有具体的 FLOPS、加速比数字，简历不空洞
  4. 可渐进 —— 一层层做，随时可以停，每层都能写进简历
